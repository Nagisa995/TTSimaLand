Затраченное время: 6ч выполнение + 1ч документация = 7 часов 

Выполненный функционал: 
Форма авторизации в три шага, точка входа и выхода не реализованы, шаги крутятся по циклу, для полного сброса цикла требуется удалить запись в localStorage с ключом "authorization".

    Реализована возможность быстрого добавления новых этапов в произвольной последовательности по средством:
    1) В компоненте App установлена сетка роутов отвечающих за этап, для добавления добавить новый роут с ссылкой на заранее подготовленный компонент (для составления компонента приготовлены шаблонные элементы) и добавить дополнительный шаг в enum STEPS записанный в файле helpers/default_value, где также размещается базовое состояние формы добавляемого компонента.
    2) В файле store/reducers/authorization в slice добавить дополнительную функцию для сохранения данных.
    3) В случае смены порядка форм, для корректной работы необходимо скорректировать порядок переключения и привязки компонентов роутах в компоненте App, в submit оброботчиках для каждой формы, которая сменила порядок и крайнии от них, а также в функциях сохранения данных в файле указанном в пункте (2).

    Реализованные шаблонные компоненты:
    1) FormContainer - общий контейнер содержимого компонента (Container компонент из MaterialUI), не принимает props, стиль предустановлен, используется как двойной тэг.
    2) FormTitle - компонент для заголовка формы, принимает обязательный prop title строчного типа, используется как одинарный тэг.
    3) FormContent - контейнер для заполняемых полей формы, принимает любые props для форм, используется как двойной тэг.
    4) CustomInput - стилизованный input для заполнения данных формы, принимает любые props по аналогии с props компонента TextArea из MaterialUI (для передачи ref используйте prop ref), используется как одинарный тэг.
    5) SubmitButton - стилизованная кнопка с type submit, принимает любые props по аналогии с props компонента Button из MaterialUI, используется как двойной тэг.

    Реализован механизм возврата пользователя на этап на котором он остановился.
